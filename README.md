## **Deskripsi Aplikasi**

**TomatCheck** adalah aplikasi yang dirancang untuk membantu pengguna memeriksa kondisi tomat—apakah masih segar atau sudah busuk—dengan cepat, teliti, dan efisien.
Dilengkapi fitur:

* **Deteksi real-time** menggunakan kamera atau unggahan gambar
* **Antarmuka sederhana dan mudah dipahami**
* **Proses analisis cepat dengan model kecerdasan buatan**
* **Informasi hasil deteksi yang akurat dan langsung ditampilkan**

Aplikasi ini membantu memastikan kualitas dan keamanan bahan makanan sehari-hari tetap terjaga.

## **Cara Menggunakan Aplikasi**

1. Buka aplikasi TomatCheck
   saat pertama kali masuk aplikasi, pengguna akan melihat 3 halaman introduction.
2. Registerasi dan Login
   Jika pengguna belum memiliki akun, silakan membuat akun dengan mengklik "Tidak punya akun? Daftar", setelah mengklik maka akan masuk kehalaman registerasi, masukan full nama, email, password, dan pilih gender lalu klik Daftar, setelah berhasil daftar maka penguna bisa masuk aplikasi.
3. halaman awal
4. cara melakukan mendeteksi Tomat busuk atau Segar 
   * Ambil foto tomat menggunakan kamera, atau Unggah foto dari galeri.
   * Muncul pop up "konfirmasi Gambar" dengan ada pilihan batal atau Prediksi Gambar.
   * Setelah menekan Prediksi Gambar Sistem akan otomatis mendeteksi tomat dan masuk ke Registerasi.
5. Tentang Aplikasi
   Pengguna dapat membaca tentang aplikasi
7. profile
   Pengguna bisa mengubah nama dari akun mereka dan juga bisa Logout

## **Cara Menjalankan Server/API**

1. Masuk ke folder backend:

   ```bash
   cd api/Synapse
   ```
2. Aktifkan environment:

   ```bash
   conda activate pakbmobilekl5
   ```
3. Jalankan server Django:

   ```bash
   python manage.py runserver
   ```

Backend akan berjalan pada `http://127.0.0.1:8000/` dan siap menerima request dari aplikasi.
